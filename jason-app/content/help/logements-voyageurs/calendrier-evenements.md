## La différence entre un séjour et un événement

- Un **séjour** = une réservation (logement + voyageur + dates + montant)
- Un **événement** = une note interne sur ton calendrier (ménage, blocage, maintenance, etc.)

Les **événements ne sont jamais visibles par les voyageurs**. Ils servent à toi pour t'organiser.

---

## Créer un événement

### Méthode rapide : clic sur une date

1. Va dans le **Calendrier**
2. **Clique** sur une date (ou clique-glisse pour sélectionner une plage)
3. Le formulaire s'ouvre, pré-rempli avec les dates

### Méthode bouton

Clique sur **Ajouter un événement** en haut du calendrier.

---

## Les champs d'un événement

- **Titre** : court et descriptif. Ex: "Ménage Martine", "Blocage travaux", "Famille en visite"
- **Date de début** et **date de fin** : pour les événements multi-jours
- **Catégorie** : Ménage, Blocage, Maintenance, Autre
- **Description** : tes notes internes (ex: "Plombier de 9h à 11h")

---

## Les catégories et leurs couleurs

Chaque catégorie a une couleur distincte pour repérer en un coup d'œil :

| Catégorie | Couleur | Cas typique |
|-----------|---------|-------------|
| **Ménage** | Vert | Nettoyage entre deux séjours |
| **Blocage** | Rouge | Dates indispos (travaux, usage perso) |
| **Maintenance** | Orange | Plombier, électricien, livraison |
| **Autre** | Gris | Tout le reste |

---

## Astuces d'utilisation

### Cliquer-glisser pour les plages multi-jours

Au lieu de remplir manuellement deux dates, **clique sur le premier jour et glisse jusqu'au dernier**. Le formulaire s'ouvre pré-rempli avec la bonne plage.

### Notes internes utiles

Le champ description est ton espace privé. Exemples concrets :

- "Ménage Martine — 09h-11h, code 1234"
- "Chauffe-eau HS, plombier confirmé Lundi 14h"
- "Réservé pour ma sœur, prévenir voisinage"

### Récurrence

La fonctionnalité de récurrence (ex: "ménage tous les dimanches") **n'est pas encore disponible**. En attendant :

- Crée un événement sur une plage multi-jours pour les blocages longs
- Crée les événements ménage un par un (compte 5 secondes par création)

---

## Modifier ou supprimer un événement

Clique sur l'événement dans le calendrier. Un menu s'ouvre :

- **Modifier** : tous les champs sont éditables
- **Supprimer** : suppression immédiate, irréversible

> Si tu cherches à **bloquer plusieurs jours** depuis Airbnb (par exemple pour rénover), il vaut mieux le faire **directement sur Airbnb** (qui le synchronisera vers le dashboard via iCal), pas l'inverse.
