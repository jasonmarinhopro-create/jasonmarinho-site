FAQ

Questions fréquentes

Le générateur de QR code est-il vraiment gratuit ?
Oui. Le générateur de QR code simple (WiFi, URL, avis Google, téléphone…) est entièrement gratuit et sans limite. L'export PNG transparent et SVG est inclus. Le générateur d'affiche A4 est disponible en plan Standard (1,98 €/mois).
Est-ce que le QR code WiFi fonctionne sur iPhone et Android ?
Oui. Le format WIFI: est supporté par l'appareil photo natif d'iOS 11+ et Android 9+ sans application supplémentaire. Le téléphone se connecte automatiquement en scannant le code.
Peut-on utiliser l'affiche directement chez l'imprimeur ?
Oui. L'export PDF est au format A4 210×297mm en haute résolution (2× : environ 1600×2250px), parfait pour l'impression en ligne (Pixartprinting, Vistaprint, imprimeur local).
Mes données WiFi sont-elles stockées ?
Non. La génération du QR code se fait entièrement dans votre navigateur. Aucune donnée WiFi n'est transmise à nos serveurs. Pour les affiches sauvegardées dans votre espace (plan Standard), les données sont liées à votre compte sécurisé.
Quelle est la différence entre le QR code simple et l'affiche A4 ?
Le QR code simple génère uniquement le code (fond transparent, à intégrer dans vos propres designs Canva, Word, Figma). L'affiche A4 génère une affiche complète avec nom du logement, WiFi, contacts et QR code, en haute résolution prête pour l'impression.
